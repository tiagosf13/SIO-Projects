To add permissions:

    <Directory "your/directory/specify">
        Order allow,deny
        Allow from all
    </Directory>

    sudo chown -R www-data:www-data your/directory/specify
    sudo chmod -R 755 your/directory/specify


To remove permissions:

    **DELETE**
    <Directory "your/directory/specify">
        Order allow,deny
        Allow from all
    </Directory>

    sudo chown -R my_username:my_username your/directory/specify
    sudo chmod -R 755 your/directory/specify


To start/stop/restart apache2:

    sudo systemctl start apache2
    sudo systemctl stop apache2
    sudo systemctl restart apache2


To verify if the website is working:

    curl -I https://financialtipeco.online


To edit the apache website:

    sudo nano /etc/apache2/sites-enabled/myflaskapp.conf


To verify the error log:

    sudo tail -n 50 /var/log/apache2/error.log


To verify the access log

    sudo tail -n 50 /var/log/apache2/access.log
